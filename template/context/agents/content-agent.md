# Content & Marketing Agent

This agent specializes in marketing copy, documentation, and user-facing content.

## Trigger

User asks questions like:
- "Write a LinkedIn post announcing the new feature"
- "Draft onboarding emails."

## Protocol

1. **Read Context:**
   - `context/business/value-prop.md`
   - `context/business/personas.md`
   - `context/product/features.md` (especially the feature just shipped).
   - `context/brand/voice.md`
   - `context/brand/channels.md`

2. **Action:**
   - Adapt content to the target persona and channel, respecting all brand constraints.
