# Providers

## Git hosting

- **GitHub** — repository, Issues, PRs.

## Agent host

- **Claude Code** — primary. Skill installed at user level (`~/.claude/skills/`) or project level (`.claude/skills/`).

## No other providers

- No databases.
- No SaaS APIs.
- No CI/CD pipelines.
- No auth providers.
- No deployment targets.

RBC's entire surface is Git + Markdown + the Claude skill.
